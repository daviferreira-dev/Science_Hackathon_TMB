		Learning Paradigm: Supervised, Unsupervised e Reinforcement
		O supervisionado chama muita atenção (classificar fpds, 0 e 1, probabilidade), trabalhar com modelos supervisionados.
		
	3 modelos q podemos usar: Deep Learning, Ensemble Methods, Regressão, o modelo deve ser de acordo com o objetivo (classificar fpds, 0 e 1, probabilidade)
	Regressão Logistica como baseline
	Cada algoritmo foca em uma coisa só, focar de novo no objetivo de classificação.

		Baseline - Começar com um modelo simples, ter o mínimo e aumentar complexidade. Agile Methods

		Fine tunning - Ajustar hiper parâmetros do modelo, acrescentando e testando parâmetros.
		Overfitting - Evitar decorar o treino. Modelo q não generaliza bem, ele começou a decorar os pontos ao invés de tracar uma linha entre os pontos.
